# clab_criptomonedas_servidor_https_spring
Servidor HTTPS Spring para la operativa CLAB (Creación, Lectura, Actualización, Borrado) con la tabla criptomonedas de la base de datos criptomonedas utilizando para ello el servicio de JDBC_servidor_HTTPS_Spring con driver JDBC de MySQL

Depende de:
- Modelos: https://github.com/EAG-es/modelos
- Modelos_comunicaciones: https://github.com/EAG-es/modelos_comunicaciones
- Modelos_html: https://github.com/EAG-es/modelos_html
- Servidor HTTPS Spring: https://github.com/EAG-es/servidor_https_spring

El servidor se es una aplicación construida con proyecto de JDBC_servidor_HTTPS_Spring: https://github.com/EAG-es/jdbc_servidor_https_spring
